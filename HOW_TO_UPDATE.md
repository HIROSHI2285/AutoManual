# 既存環境のアップデート手順

別のPCですでにAutoManualの環境を構築している場合、最新の機能（縦2行レイアウトや画像ZIP一括保存など）を取り込むためのアップデート手順をご案内します。

以下の手順に沿って、最新版のコードの取得と必要な追加パッケージのインストールを行ってください。

## 1. 起動中の開発サーバーを停止する

現在動かしている開発サーバー（`npm run dev`）がある場合は、一度終了させます。
黒い画面（コマンドプロンプトやPowerShellなど）で以下のキーを押してください。

**`Ctrl` + `C`**

「バッチ ジョブを終了しますか (Y/N)?」と表示されたら、**`y`** と入力して `Enter` キーを押します。

## 2. GitHubから最新のプログラムを取得する

AutoManualがインストールされているフォルダ（例: `C:\Users\...\AutoManual`）でコマンドラインを開き、以下のコマンドを実行して最新のコードをダウンロード（プル）します。

```bash
git pull
```

> **注意**: もし手元で直接コードを変更してしまっていて競合（コンフリクト）エラーが出た場合は、一時的に変更を退避するか破棄する必要があります（例: `git stash`）。

## 3. 追加パッケージをインストールする（重要）

新機能の追加（特にZIPファイルの一括作成機能など）に伴い、新しいシステムパッケージが追加されています。
コードを最新にした直後に、必ず以下のコマンドを実行して必要なライブラリを追加インストールしてください。

```bash
npm install
```

> **重要**: この手順をスキップすると、「モジュールが見つかりません」といったエラーが発生し、アプリが起動しない原因となります。

## 4. 開発サーバーをもう一度起動する

追加インストールが無事に終わったら、いつも通りアプリを立ち上げ直します。

```bash
npm run dev
```

ブラウザで [http://localhost:3000](http://localhost:3000) を開き、最新のアプリ画面が表示されればアップデート完了です！
